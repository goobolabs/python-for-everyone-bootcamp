# Library Management System

## Overview
Library Management System is a Python console application used to manage library books. The system allows users to add, view, search, update, delete, and save book records.

## Features
- Add new books
- View all books
- Search for books
- Update book details
- Delete books
- Save books to a file
- Exit

## Project Structure

```text
LIBRARY_MANAGEMENT/
│
├── data/
│   └── books.txt
│
├── models/
│   ├── __init__.py
│   └── books.py
│
├── utilities/
│   └── storage.py
│
└── main.py
```


## Requirements

- Python 3.x

## Running the Application

Open a terminal and run:

```bash
python main.py
```


## Menu


Library Management System

##  Workflow

1. Add a new book.
2. View all available books.
3. Search for a specific book.
4. Update book information if needed.
5. Delete unwanted books.
6. Save all changes.
7. Exit the application.
