# Store Manager — Final Project

**Student:** Ali Salad  
**Bootcamp:** Python for Everyone  
**Assignment:** 07 — Final Project  

## Description

This project is a command-line Inventory and Staff Management System built in Python 3.10+.
It allows a store owner to manage products and employees directly from the terminal.
The system supports adding, updating, searching, restocking, and removing products,
as well as hiring, updating, and firing staff members. All data is saved to text files
so it persists between sessions.
