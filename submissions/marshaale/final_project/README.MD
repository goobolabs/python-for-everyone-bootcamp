# Car Rental CLI Project

A simple command-line car rental application built with Python.

This project was created to practice and demonstrate the skills I learned in the Python for Everyone Bootcamp by [Goobo Labs](https://github.com/goobolabs).
<br>

The source code for this project is available on [Project](https://github.com/marshaale/car-rental-python-cli-project)

## Features

This project covers:

- Python fundamentals
- File handling
- Object-oriented programming (OOP)
- Path management
- Classes and objects
- Other core Python concepts and best practices
