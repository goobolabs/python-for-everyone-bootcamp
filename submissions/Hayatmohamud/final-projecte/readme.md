# Human Resource Management System

## Overview

The **Human Resource Management System (HRMS)** is a Python console application developed as a final project. It allows users to manage employee records through a simple menu-driven interface. The system demonstrates the use of Python functions, lists, dictionaries, file handling, and basic CRUD (Create, Read, Update, Delete) operations.

---

## Features

* Add a new employee
* View all employees
* Search for an employee by ID or name
* Update employee information
* Remove an employee
* Save employee data before exiting

## How the System Works

### 1. Add Employee

Allows the user to enter new employee information such as Employee ID, Name, Position, Salary, or other details.

### 2. List Employees

Displays all employee records currently stored in the system.

### 3. Search Employee

Finds an employee using their ID or name.

### 4. Update Employee

Edits the information of an existing employee.

### 5. Remove Employee

Deletes an employee record from the system.

### 6. Save and Exit

Saves all employee data and safely exits the application.
