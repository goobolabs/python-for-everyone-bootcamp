Inventory Stock Management System

Overview

Inventory Stock Management System is a Python application that helps users manage product inventory efficiently. The system allows users to add, update, search, view, and remove products while storing inventory information for future use.

Features

* Add new products
* View inventory items
* Update product information
* Search for products
* Remove products from inventory
* Store inventory data in files
* Error handling for invalid inputs

Technologies Used

* Python 3
* Object-Oriented Programming (OOP)
* File Handling
* Modular Programming

Project Structure

inventoryStock_mangement_system/

├── utils/

├── .gitignore

How to Run

1. Clone the repository:
    git clone https://github.com/fatima-moha143/inventoryStock_mangement_system.git
2. Navigate to the project folder:
    cd inventoryStock_mangement_system
3. Run the program:
    python main.py

Reflection

In this project, I developed an Inventory Stock Management System using Python. I applied Object-Oriented Programming concepts to organize the code and make it easier to maintain. The project structure was divided into separate modules, including models, utilities, and the main application file.

File handling was used to store and retrieve inventory information. Error handling was implemented to prevent the application from crashing due to invalid input or missing files. Separating the project into different folders improved readability and code organization.

This project helped me improve my understanding of Python programming, OOP principles, file handling, debugging, and modular software development. In the future, I would like to add database support, a graphical user interface, and advanced inventory reporting features.
