# Movie Recommendation System

A Python CLI-based movie recommendation system with:

- User authentication (Login/Registration)
- View All Movies
- Search Movies
- Get Recommendations
- Rate a Movie
- My Favorites
- Watch History
- Admin: Manage Movies
- Reports & Statistics
- Save/load data from JSON files

## Install

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```
