# 🐍 Python Quiz App

##  Project Description

Python Quiz App is a simple command-line application built with Python. The program allows users to test their basic Python knowledge by answering multiple-choice questions. At the end of the quiz, the application calculates and displays the final score.

---

##  Features

* Interactive menu
* Start or exit the quiz
* Multiple-choice questions
* Instant feedback for each answer
* Automatic score calculation
* Final score displayed after completing the quiz

---

##  Technologies Used

* Python 3
* Visual Studio Code (VS Code)
* Terminal / Command Prompt

---

##  Quiz Topics

The quiz includes questions about:

* Python loops
* User input
* Comments
* Conditional statements

---

##  How to Run the Project

1. Make sure Python 3 is installed.
2. Download or clone this repository.
3. Open the project in VS Code.
4. Run the program using:

```bash
python quiz.py
```
