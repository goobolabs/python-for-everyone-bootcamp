Project Name: Airport Management System
A command-line(CL) Airport Management built with Python using Object-Oriented Programming (OOP), file handling, functions, loops, and modular programming principles.

Project Purpose
The purpose of this project is to help users manage and track their daily flaights directly from the terminal.
