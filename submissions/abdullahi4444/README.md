# Abdullahi Abdiweli Adam

## Python for Everyone Bootcamp — Submission Overview

This repository contains the bootcamp submission folder for the Python for Everyone course, including foundational assignments and the final professional POS system project.

### Repository Structure

| Folder | Description |
| --- | --- |
| `assignment1/` | First assignment with the initial quiz and practice code. |
| `assignment 2/` | Second assignment exercises and supporting notes. |
| `assignment 3/` | Third assignment work with catalog and inventory examples. |
| `Final project/` | Final capstone POS application and its professional README. |

### Final Project

The `Final project/` folder contains the complete Point of Sale application, including:

- `main.py` — application entry point
- `products.txt` — product inventory data
- `sales.txt` — transaction log
- `README.md` — detailed project documentation and acknowledgements

**Project Repository:** https://github.com/abdullahi4444/POS-System
