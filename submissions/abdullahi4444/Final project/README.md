# POS System (Point of Sale)

A professional command-line Point of Sale system implemented in Python. This application demonstrates core software design, inventory management, sales tracking, and user-centered CLI interaction — making it a strong capstone submission for the "Python for Everyone" bootcamp.

## Project Link

- Repository: https://github.com/abdullahi4444/POS-System

## Key Features

- Add new products, edit existing inventory, and view product listings
- Build a customer cart with product selection and quantity control
- Calculate totals and finalize sales with simple checkout logic
- Persist all sale records to `sales.txt` for review and auditing
- Use `products.txt` as the data source for product inventory

## How to Use

1. Start the application.
2. Select from the menu to manage products or start a sale.
3. Add items to the cart and confirm checkout.
4. Completed transactions are saved automatically to `sales.txt`.
