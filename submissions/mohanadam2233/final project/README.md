# Inventory Management CLI

A Python CLI-based inventory management system with:

- Admin login
- Add product
- View products
- Search product
- Update product
- Delete product
- Save/load products from JSON
- Low stock warning
- Product categories
- Sell product
- Sales history
- Profit calculation

## Install

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```
